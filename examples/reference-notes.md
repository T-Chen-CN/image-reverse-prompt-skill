# Reference Notes

This example is for analyzing a group of related reference images.

## Basic flow

1. Summarize the image group.
2. Describe the role of each reference.
3. Identify repeated visual patterns.
4. Separate stable rules from changeable fields.
5. Return a reusable final structure.

## Stable rules

- clear subject focus
- consistent lighting
- simple background
- controlled color system
- user-confirmed facts stay unchanged

## Changeable fields

- subject
- format
- color
- copy
- scene
- usage purpose
