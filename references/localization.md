# Localization

This Skill is market-neutral by default.

## Default behavior

- Do not assume the target country.
- Do not assume the target platform.
- Do not assume the target language.
- Use the user's language unless they request another language.

## When the user provides a market or platform

Adapt:

- copy length
- visual tone
- phrasing style
- product claim strictness
- text hierarchy
- marketplace-safe wording

## Text in images

If image text is needed:

- keep it short
- use user-provided copy when available
- do not translate claims into stronger claims
- do not add certifications, specifications, or performance promises not provided by the user
- list uncertain text separately instead of treating it as fact

## Variable fields

Use variables for localizable copy:

- `{{target_language}}`
- `{{main_headline}}`
- `{{subtitle}}`
- `{{small_text}}`
- `{{feature_labels}}`
- `{{cta_text}}`
